Manual del Usuario

Guia del usuario del Punto de Venta CISK

Introducción

Bienvenido al Punto de Venta CISK. Esta aplicación le ayudará a hacer las ventas de Shekapan eficientemente.

Nota: Esta versión es incial, por lo que mejoras eventuales serán implementadas eventualmente... (Versión 1.0?)

Inicio de Sesión

Step 1: Installation

Download the installer from the official website and run the setup wizard.

Step 2: Configuration

  • Open the settings menu.
  • Enter your API key.
  • Save changes.

Atajos de Teclado

Los atajos de teclado más comunes son:

Función Descripción Atajo
Validar con Efectivo Valida venta y certifica con SAT recibiendo Efectivo como forma de pago. F10
Imprimir Ticket de Venta Valida la venta e imprime ticket recibiendo Efectivo como forma de pago. Ctrl + F10
Tarjeta de Crédito Valida la venta y certifica con SAT recibiendo tarjeta de débito o crédito como forma de pago. F9
Tarjeta de Crédito con Ticket Valida la venta e imprime ticket recibiendo tarjeta de débito o crédito como forma de pago Ctrl + F9
