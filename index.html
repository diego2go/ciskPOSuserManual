Manual del Usuario

Guia del usuario del Punto de Venta CISK

Introducción

Bienvenido al Punto de Venta CISK. Esta aplicación le ayudará a hacer las ventas de Shekapan eficientemente.

Nota: Esta es una versión beta...? (Versión 1.xx, 07-2026?)

Inicio de Sesión

Paso 1: Ingresar a CISK

Ingrese desde su navegador preferido a CISK

Paso 2: Iniciar Punto de Venta

En el menu lateral Ventas pulse el botón Punto de Venta. Si no ve este botón, presione la flecha junto al botón Ventas para desplegar el menú.

Ventas

Ingrese el NIT

Inicie una venta nueva ingresando el NIT del cliente. Si no tiene NIT, puede omitir este paso presionando Enter.

Ingrese cantidad de productos

  • Ingrese la cantidad de productos.
  • Pulse el botón Enterpara saltar a la siguiente línea de producto.
  • Ingrese la cantidad de productos de una nueva línea.

Nota: Si no conoce el código del producto a cobrar, puede buscar en el campo Artículo: mientras escriba, CISK le mostrará los productos que coinciden con su término de búsqueda. Seleccione con el ratón o las flechas de teclado el artículo que desea facturar.

Finalizar Venta

Seleccione la forma de pago del menú lateral derecho. Una ventana emergente le muestra el resumen de la venta y la opción de ingresar el monto recibido. Al ingresar en el campo recibido, CISK calcula el cambio automáticamente.

Nota: Use los atajos de teclado descritos en la sección Atajos de Teclado para cobrar rápidamente.

Reimpresión de Facturas

  • Pulse el botón Listado o use el atajo F7.
  • Busque en el listado la factura que desea reimprimir. Puede usar el campo de búsqueda si conoce el nombre o NIT del cliente.
  • Pulse el botón 🖨️ para imprimir el documento.

Anulación de Facturas

  • Pulse el botón Listado o use el atajo F7.
  • Busque en el listado la factura que desea anular. Puede usar el campo de búsqueda si conoce el nombre o NIT del cliente.
  • Pulse el botón de ❌ para anular el documento.

Notas y observaciones de Tickets

Para agregar referencias u observaciones en la venta, anótelas en el campo Referencia / Observaciones justo debajo del Total a Pagar.

Abrir Gaveta

Para abrir la gaveta de dinero, pulse el botón Gaveta del menú lateral.

Tip: Use el atajo de teclado Ctrl + F11 para abrir la gaveta de dinero rápidamente.

Búsqueda de Clientes

Para realizar una búsqueda avanzada de clientes, pulse el botón Buscar clientes del menú lateral.

Tip: Use el atajo de teclado F2 para ingresar a la búsqueda avanzada de clientes.

Busque por NIT, nombre o DPI. Al encontrar el cliente deseado, seleccione con el mouse o con las flechas y presione Guardar. La información del cliente se cargará a una nueva venta del Punto de Venta.

Menu Administrativo

Presione el botón Menú administrativo del menú lateral.

Atajos de Teclado

Los atajos de teclado más comunes son:

Función Descripción Atajo
Pago en Efectivo Valida venta y certifica con SAT recibiendo Efectivo como forma de pago. F10
Imprimir Ticket de Venta Valida la venta e imprime ticket recibiendo Efectivo como forma de pago. Ctrl + F10
Tarjeta de Crédito Valida la venta y certifica con SAT recibiendo tarjeta de débito o crédito como forma de pago. F9
Tarjeta de Crédito con Ticket Valida la venta recibiendo tarjeta de débito o crédito como forma de pago Ctrl + F9
Ticket con Múltiples Formas de Pago Valida la venta e imprime ticket recibiendo tarjeta de débito o crédito y efectivo como formas de pago F8
Listado Abre un listado con las facturas emitidas. F7
Gaveta Abre la gaveta de dinero. Ctrl + F11
Buscar clientes Abre un menú de búsqueda avanzada de clientes. F2

Problemas Frecuentes

Problema Descripción Solución
Error 401 El punto de venta se bloquea en la pantalla de alerta con el Error 401. Verifique la conexión a Internet. Reinicie el navegador (Alt + F4) y abra el acceso directo CISK Punto de Venta.
Código inválido de producto Valida la venta y certifica con SAT recibiendo tarjeta de débito o crédito como forma de pago. F9
Otros Problemas Describir aqui Ctrl + F9
